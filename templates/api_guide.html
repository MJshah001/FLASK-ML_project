API Guide - Flight Price Prediction

This guide will help you understand how to interact with the API endpoint.

API Endpoint
https://flask-ml-project-flight-price-prediction.onrender.com/api/predict
Request Format

The request should have the following format and include the Content-Type: application/json header:


{
    "airline": "Air India",
    "date_of_journey": "2024-08-16",
    "source": "Delhi",
    "destination": "Cochin",
    "dep_time": "13:20:00",
    "arrival_time": "15:20:00",
    "duration": 2000,
    "total_stops": 10,
    "additional_info": "Business Class"
}
Inputs

Here are the possible values and formats for each input field:

  • Airline:

    • Air India
    • Jet Airways
    • Indigo
    • Multiple Carriers
    • Spicejet
    • Vistara
    • Air Asia
    • Goair
    • Trujet

  • Date of Journey:

    Format: YYYY-MM-DD

  • Source:

    • Delhi
    • Kolkata
    • Banglore
    • Mumbai
    • Chennai

  • Destination:

    • Cochin
    • Banglore
    • New Delhi
    • Delhi
    • Hyderabad
    • Kolkata

  • Departure Time:

    Format: HH:MM:SS

  • Arrival Time:

    Format: HH:MM:SS

  • Duration:

    Format: Integer (in minutes)

  • Total Stops:

    Format: Integer

  • Additional Info:

    • No Info
    • In-flight meal not included
    • 1 Long layover
    • No check-in baggage included
    • Change airports
    • Red-eye flight
    • Business class
    • 1 Short layover
    • 2 Long layover

Response Format

The API will return a JSON response with the predicted flight price. An example response might look like this:


{
    "prediction": 13435.7
}

Ensure that all fields are correctly formatted as per the example provided. Invalid data or missing fields may result in errors.

Important: This API is for demonstration purposes only and may not be suitable for production use. The model used for prediction is a simple machine learning model and may not be accurate for all scenarios.
